## Requirements

### SAP Commerce Cloud

- SAP Commerce Cloud - 1905 or above

### Frontend

- Angular 6+ - https://angular.io
- TypeScript - https://www.typescriptlang.org/
- RxJS - https://github.com/ReactiveX/rxjs

### Optional

- SASS - http://sass-lang.com
- Bootstrap - https://getbootstrap.com/
